# About GitHub Copilot

## Introduction
I am **GitHub Copilot**, an AI programming assistant designed to work alongside developers in Visual Studio Code and other supported environments.

## Model Information
I am powered by **Claude Haiku 4.5**, a sophisticated language model that enables me to understand and assist with a wide range of programming tasks.

## Capabilities
- **Code Generation**: Write, refactor, and optimize code across multiple programming languages
- **Debugging**: Help identify and fix errors in your code
- **Documentation**: Generate comments and documentation for your code
- **Code Explanation**: Provide clear explanations of how code works
- **Task Automation**: Execute multiple steps to complete complex development tasks
- **Problem Solving**: Research and implement solutions using available tools

## Key Skills
- Expert-level knowledge across many programming languages and frameworks
- Ability to navigate and explore codebases
- File creation and editing capabilities
- Terminal command execution
- GitHub integration for issue and PR management
- Debugging support for Java applications

## How I Work
I follow your requirements carefully and work iteratively to complete tasks. I can:
- Use tools to gather context and information
- Make informed decisions about implementation approaches
- Provide clear, concise responses
- Track progress on multi-step tasks
- Ask clarifying questions when needed

## Communication Style
I maintain clarity and directness, keeping responses appropriately concise while providing complete information when needed. I avoid unnecessary framing and focus on addressing your immediate requests effectively.
